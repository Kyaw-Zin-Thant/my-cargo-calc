# AI Smart Voucher Calculator ⚡🛒

Gemini 2.5 Flash စနစ်သုံး ဝယ်ကုန်ပစ္စည်းများနှင့် ကာဂိုခ (Cargo Fees) များကို ပြေစာ/ဘောက်ချာပုံရိပ်များမှတစ်ဆင့် အလိုအလျောက် တိကျစွာ တွက်ချက်ပေးမည့် Smart Web Tool ဖြစ်ပါသည်။ 

မြန်မာနိုင်ငံသို့ ကာဂိုဖြင့် ပစ္စည်းပြန်သွင်းသူများ၊ Online Shop ပိုင်ရှင်များနှင့် Mart ဘောက်ချာများအား ကျပ်ငွေဖြင့် ရောင်းဈေးတွက်ချက်လိုသူများအတွက် အထူးသင့်လျော်ပါသည်။

## ✨ ထူးခြားချက်များ (Features)
- **AI OCR Integration:** Gemini 2.5 Flash ကို အသုံးပြုထားသဖြင့် ရှုပ်ထွေးသော MM Mega Market ဘောက်ချာများနှင့် Shopee Screenshots များမှ Data များကို ၁၀၀% တိကျစွာ ခွဲထုတ်ပေးနိုင်ခြင်း။
- **Unit Price Detection:** စုစုပေါင်း ကျသင့်ငွေနှင့် တစ်ထုပ်ချင်းစီ၏ မူရင်းရင်းဈေး (Unit Price) ကို မမှားမယွင်းစေဘဲ သီးသန့် ခွဲခြားဖတ်ရှုပေးနိုင်ခြင်း။
- **Dynamic Calculation:** ငွေလဲနှုန်း (VND/MMK)၊ ၁ ကီလိုဂိုဒေါင်ခ (Cargo Rate) နှင့် အမြတ်ရာခိုင်နှုန်းတို့အပေါ် မူတည်ပြီး အဝိုင်းကိန်း ရောင်းဈေးများကို စက္ကန့်ပိုင်းအတွင်း တွက်ချက်ပေးခြင်း။
- **Interactive Table:** AI မှ ဖတ်ပေးလိုက်သော အရေအတွက် (Qty) နှင့် အလေးချိန် (Grams) များအား ဇယားကွက်ထဲတွင် တိုက်ရိုက် ပြန်လည်ပြင်ဆင်နိုင်ခြင်း။
- **Export to PDF:** ခလုတ်တစ်ချက်နှိပ်ရုံဖြင့် စာရင်းဇယားများအား သန့်ရှင်းသပ်ရပ်သော PDF / Print Slip အဖြစ် ထုတ်ယူနိုင်ခြင်း။
- **Privacy First & Free:** No Billing / Free Tier ဖြင့် အသုံးပြုနိုင်ပြီး ဒေတာများအား LocalStorage ပေါ်တွင်သာ သိမ်းဆည်းပေးခြင်း။
